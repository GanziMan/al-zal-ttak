# 알공딱 제품 개선 제안

## 현재 서비스 진단

**잘 되어 있는 것:** AI 분석 파이프라인, 카카오 인증, 관심종목/북마크/설정, 텔레그램 알림, 기업 재무 상세
**가장 큰 약점:** 유저가 들어와서 "뭘 해야 하지?"를 모름. 온보딩 부재 + 재방문 동기 부족

---

## 1. 리텐션 — 지금 가장 급한 것

| 개선 | 왜 필요한가 | 난이도 |
|------|------------|--------|
| **오늘의 AI 브리핑** (데일리 요약) | 매일 아침 "어제 공시 중 주목할 것 3건"을 한 화면에. 재방문 이유가 됨 | 중 |
| **Web Push 알림** | 텔레그램은 진입장벽이 높음 (봇 설정 필요). SW가 이미 등록되어 있으니 Web Push만 추가하면 됨 | 중 |
| **주간 리포트 이메일** | "이번 주 관심종목 공시 5건, 호재 2건" 같은 요약 메일. 서비스 밖에서 리마인드 | 중 |

---

## 2. 온보딩 — 신규 유저 이탈 방지

| 개선 | 왜 필요한가 | 난이도 |
|------|------------|--------|
| **인기 종목 추천** | 가입 직후 관심종목이 0개 → 대시보드가 텅 빔. "많이 추적하는 종목 TOP 10"이나 "코스피 시총 상위 20" 원클릭 추가 | 하 |
| **업종별 그룹핑** | 삼성전자, SK하이닉스를 하나하나 검색 추가하는 게 아니라 "반도체" 클릭으로 한번에 | 중 |
| **공시 용어 툴팁** | "유상증자", "전환사채", "자기주식처분" 같은 용어에 말풍선 설명. 초보자 진입장벽 해소 | 하 |

---

## 3. 분석 깊이 — 차별화 포인트

| 개선 | 왜 필요한가 | 난이도 |
|------|------------|--------|
| **기업별 공시 타임라인** | 회사 상세 페이지에 "최근 3개월 공시 흐름"을 시각화. 연속된 악재/호재 패턴이 보임 | 중 |
| **포트폴리오 영향도** | "내 관심종목 전체 기준, 이번 주 호재/악재 비율"을 점수화. 투자 심리 온도계 역할 | 중 |
| **공시 vs 주가 연동** | 공시 발표일 전후 주가 변동을 같이 보여주면 AI 분석의 신뢰도가 체감됨. 한국투자증권/키움 API 또는 네이버 금융 크롤링 | 상 |
| **비슷한 과거 사례** | "이 공시와 유사한 과거 공시에서 주가는 +3.2% 상승했습니다" — 현재 keyword 기반 유사공시가 있지만 너무 단순 | 상 |

---

## 4. 소셜 & 바이럴

| 개선 | 왜 필요한가 | 난이도 |
|------|------------|--------|
| **공시 카드 공유** | 분석 결과를 카카오톡/트위터에 이미지 카드로 공유. OG 이미지가 이미 있으니 공시별 동적 OG로 확장 | 중 |
| **AI 분석 한줄평 투표** | "이 분석에 동의하시나요?" 👍/👎 → AI 정확도 피드백 루프 + 유저 참여감 | 하 |
| **실시간 핫 공시** | "지금 가장 많이 조회되는 공시" 랭킹 → 비로그인 유저도 흥미를 느낌 | 중 |

---

## 5. 수익 모델 (장기)

| 방향 | 설명 |
|------|------|
| **프리미엄 티어** | 무료: 관심종목 5개, 7일 조회 / 프리미엄: 무제한 종목, 30일+, 실시간 알림, 주간 리포트 |
| **기업 IR용 대시보드** | 상장기업 IR팀에게 "우리 회사 공시가 AI에게 어떻게 분류되는지" 보여주는 B2B 도구 |
| **API 제공** | 핀테크/로보어드바이저 업체에 공시 분석 API를 유료 제공 |
